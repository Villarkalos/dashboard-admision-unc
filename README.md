# 📊 Dashboard Admisión UNC en Cifras

## 🧠 Contexto
¿Qué hace que un reporte de Power BI sea realmente profesional?

Este proyecto responde esa pregunta mediante el desarrollo de un dashboard analítico sobre los procesos de admisión de la Universidad Nacional de Cajamarca (UNC).

---

## 🎯 Problema de negocio
Las universidades gestionan grandes volúmenes de datos en procesos de admisión:
- ¿Qué carreras son más demandadas?
- ¿Dónde existe mayor competencia?
- ¿Cómo varía la tasa de admisión entre sedes?

Sin visualización, estas respuestas permanecen ocultas en datos tabulares.

---

## ⚙️ Solución desarrollada

### 🔹 Fuente de datos
Dataset estructurado con:
- Examen
- Carrera
- Modalidad
- Postulantes
- Ingresantes
- Fecha

---

### 🔹 Transformación (Power Query)
- Limpieza de nombres de carrera
- Extracción de sede desde texto
- Creación de columnas de tiempo (Año, Mes, Periodo)
- Coordenadas geográficas por sede

---

### 🔹 Modelo DAX (8 capas analíticas)

1. KPIs principales  
2. Análisis temporal  
3. Rankings dinámicos  
4. Competitividad  
5. Análisis por sede  
6. Dispersión estadística  
7. Textos dinámicos  
8. HTML personalizado  

---

## 🎨 Innovación clave
Uso de HTML dentro de Power BI mediante DAX para:

- Navegación entre páginas
- KPIs personalizados
- Glosario integrado
- Títulos dinámicos
